# ProHealth - Medical & Healthcare Website

## Description
This is a multi-page medical and healthcare website created from a Figma design using **Gemini CLI**.  
The project showcases a static layout with **10 pages**, styled using plain **HTML** and **CSS**.

This homework was assigned in **Class 7 by Sir Jawwad Ali** and completed on **Sunday, 2–5 PM**.

---

## Features
- Functional **Navbar** and **Footer** layout  
- **10 static pages** designed from Figma  
- Styled using **HTML & CSS**  
- Fully **browser-ready**  
- Easy to extend with additional content or pages  

---

## Tools Used
- **Figma** – for designing the website  
- **Gemini CLI** – to convert Figma frames to HTML & CSS  
- **VS Code** – code editor  
- **Browser** – to view the website  

---

## How to Run
1. **Download or clone the repository**  
2. **Open the project folder**  
3. **Double click on `index.html`** (or any other page)  
4. The page will open in your default browser  

> No installation is required as this project is built with plain HTML & CSS.
